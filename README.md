# Alarm Clock App

A windows form application that allows you to make / remove Alarms, Timers and Stopwatches.

# Installation

Go to the [Release](https://github.com/RefinedDev/Alarm-Clock-App/releases/latest) Page  to install. (**Windows Only**)
